# 订你所想 在线电影订票软件

[![Build Status](https://travis-ci.org/Kinpzz/movie-booking.svg?branch=master)](https://travis-ci.org/Kinpzz/movie-booking)

## 构建环境
Eclipse + Maven + Spring-boot

## 运行
方法一：使用maven在命令行
```
mvn spring-boot:run
```
方法二：Eclipse环境下
Run as -> Maven Build -> 在 Goals 写
```
spring-boot:run
```
